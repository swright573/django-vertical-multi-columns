The VMC Example Site

This site will show you the various VMC views in action and how to code them.

  • Each sample view includes the same 45 items, each displayed in a different configuration depending on the view.

    • Look at views.py to see how each was coded.

    • Look at the templates to see how the view data is referenced for display.

  • You can configure the default number of columns in settings or you can specify this in your VMC view. In priority order:

    • Pass the number of columns as kwarg 'num_columns'. (See class EvenVMC in views.py in the example app.)

    • Define the default number of columns in your Django settings. (See settings.py in the example app.)

    • If no kwarg is passed and there is no setting defined, the number of columns defaults to 3.
